# K-MER EVENT BOOKING SYSTEM

A premium enterprise-grade event booking platform for Cameroon, combining modern UI/UX, secure API design, and scalable full-stack architecture.

## Architecture

- `backend/` - Node.js + Express REST API
- `frontend/` - React + Vite SPA
- `DATABASE-SCHEMA.md` - relational schema overview
- `API-CONTRACTS.md` - REST endpoint contract definitions
- `DEPLOYMENT.md` - deployment guidelines for Vercel/Railway

## Quick start

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Features

- JWT authentication and role-based security
- Event discovery, filtering, and booking flow
- QR code ticket generation
- Admin dashboard with analytics
- Modern ticketing UI designed for festival and nightlife experiences
